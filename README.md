# 尼腓的家價格頁

這個資料夾可以直接上傳到 GitHub Pages 使用。

## 檔案內容

- `index.html`：價格頁主頁
- `style.css`：頁面樣式
- `images/room-price.jpg`：只訂房間價格
- `images/whole-house-price.jpg`：包棟價格
- `images/rooms-info.jpeg`：只訂房間介紹
- `images/whole-house-info.jpeg`：包棟介紹

## GitHub Pages 設定

1. 到 GitHub 建立一個新的 repository。
2. 上傳這個資料夾裡的所有檔案。
3. 到 repository 的 `Settings`。
4. 左側點 `Pages`。
5. `Build and deployment` 選：
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. 按 `Save`。

完成後會得到類似這樣的網址：

`https://你的帳號.github.io/你的repository名稱/`
